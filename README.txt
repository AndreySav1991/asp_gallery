Базова специфікація сайту : Art Gallery “Xmelinin”
Функціонування даного рішенняпризначена для управліннядинамічним веб-сайтомівключає:
•	Можливість віддаленог оадміністрування через вебінтерфейс;
•	Управлінняправами на редагування інформації;
•	Наповнення сайту через простій і зрозумілий вебінтерфейс. Він дозволяє додавати та редагувати картини,назви, та всю інформацію про них;
•	Пошук по галереї;
•	Зрозуміле та зручне меню навігації по сайту;
•	Можливість створення заказів на певну картину;
•	Зручна обробка заказів;
•	Цікавий та не навязчивий інтерфейс;
•	Є можливість багатомовності;
•	Безпека сайту – оптимальна;


Для створення цього сайту використовувалося ASP.NET/ASP.MVC. Для відображення картин та галереї - CSS, jquery ,AJAX.
Присутні 3 ролі: Адміністратор та користувач,а також гість.
Адміністратор може адмініструвати сайт, додавати нові картини, редагувати, обробляти замовлення.
Гість може переглядати картини, робити замовлення.
